# Proyek : Di Sekitar Amerika Serikat

1. **Deskripsi**

   Merupakan sebuah web yang dimana intinya adalah menambahkan sebuah gambar sesuai dengan inputan yang di kirim oleh user, yang awalnya menyediakan tampilan gambar kartu sebanyak 6 buah. Kemudian di bagian tombol tambah ada fitur validasi yang mana, user harus menginput sesuai ketentuan yang ada, jika tidak maka pesan error, dan tombol akan nonaktif. Selain itu juga user bisa mengganti nama dan pekerjaan yang bisa di klik di tombol edit dan juga memiliki fitur yang sama dengan tambah kartu. 2. **Teknologi**

   Teknologi yang digunakan untuk web ini menggunakan html, css, dan javascript murni. Kemudian untuk metodologi css menggunakan BEM _(Block Element Modifier)_. Kemudian untuk javascriptnya menggunakan tipe modul dan paradigma OOP _(Object Oriented Programming)_.
